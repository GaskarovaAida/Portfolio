### 💸 A/B-тест SollmaFin (fintech)
Анализ эксперимента по новому онбордингу инвестиционного приложения: влияние подробной информации о рисках активов на вовлечённость и монетизацию.

**Что сделано:**
- EDA исторических данных: воронка событий, привлечение по странам и платформам
- Влияние уровня риска актива на конверсию во второй депозит
- Проверка корректности A/B-теста: хи-квадрат по группам, пересечения пользователей
- Ключевая метрика ARPU: t-тест Уэлча + динамика p-value по дням эксперимента
- Барьерная и вспомогательные метрики: z-тест пропорций
- Bootstrap-анализ суммы депозитов платящих пользователей

**Стек:** `Python` `pandas` `numpy` `scipy` `matplotlib` `seaborn` `Bootstrap` `z-тест`  
🔗 [GitHub](https://github.com/GaskarovaAida/Portfolio)
