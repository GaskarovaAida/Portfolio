# Яндекс Книги — анализ активности пользователей

## Описание

Анализ активности пользователей сервиса Яндекс Книги в Москве и Санкт-Петербурге на реальных данных (сен–дек 2024). Цель — проверить гипотезу о том, что пользователи СПб проводят в приложении больше времени, чем москвичи.

---

## Задачи

1. Предобработка данных: проверка дубликатов, удаление технических столбцов
2. Сравнение активности пользователей Москвы и Санкт-Петербурга: статистики и распределения
3. Проверка гипотезы: односторонний t-тест Уэлча
4. Формирование выводов и рекомендаций

---

## Инструменты

- **Python** — основной язык анализа
- **pandas, NumPy** — обработка и агрегация данных
- **SciPy** — статистические тесты (t-тест Уэлча)
- **Matplotlib, Seaborn** — визуализация распределений

---

## Результаты

- Среднее время активности в СПб — 11,59 ч, в Москве — 10,88 ч
- Статистически значимых различий не выявлено: p-value = 0,2182
- Нулевая гипотеза не отвергается — разница может быть случайной
